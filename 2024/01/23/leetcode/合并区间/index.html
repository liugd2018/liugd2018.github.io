合并区间

2024-01-23 | leetcode

以数组 intervals 表示若干个区间的集合，其中单个区间为 intervals[i] = [starti, endi] 。请你合并所有重叠的区间，并返回 一个不重叠的区间数组，该数组需恰好覆盖输入中的所有区间 。

示例 2：

提示：

1 <= intervals.length <= 104
intervals[i].length == 2
0 <= starti <= endi <= 104

思路

首先把数组的开区间数值按照生序的方式排序，生成排序后的数据。使用当前数据的闭区间和下个数组的开区间进行比较。如果闭区间大于开区间说明下个数组也在当前区间内，闭区间小于开区间说明下个数组是在当前区间以外。
